# Profile data templates

Put your personal context files in `profile-data/` (outside this `templates` folder).

This repo keeps templates in git, but your real data stays ignored by `.gitignore`.

## Suggested files

- `profile-data/profile.md`
- `profile-data/answers.txt`
- `profile-data/resume.pdf`
- `profile-data/cover-letter.md` (optional)

## profile.md example

```text
Full Name: Jane Doe
First Name: Jane
Last Name: Doe
Email: jane@example.com
Phone: +1 555 987 6543
Location: New York, NY
LinkedIn: https://linkedin.com/in/janedoe
Github: https://github.com/janedoe
Website: https://janedoe.dev
Work Authorization: Yes
Visa Sponsorship: No
Salary Expectation: Open to market range
```

## answers.txt example

```text
Question: Why do you want to work here?
Answer: I want to contribute to high-impact developer tooling and help teams ship faster with reliable systems.

Question: Are you authorized to work in the United States?
Answer: Yes.

Notice period :: Two weeks
Preferred location :: New York City
```
